Class: IndexedDB

IndexedDB(configurationopt)

The IndexedDB Adapter provides the Space Class the ability to interact with the IndexedDB API found in most modern browsers.

Constructor

new IndexedDB(configurationopt)

Create a new IndexedDB. Differently from Local and Session Storages, the IndexedDB Adapter requires a mandatory name, version and store name.
Parameters:
Name Type Attributes Default Description
configuration Object <optional>
 {name = '', version = '', store = '', props = {}, index = {}} Configuration Object for the Adapter
Properties
Name Type Description
name string Name of the Space
version string Version of the Space in Semantic versioning syntax
store string Name of the Object Store to use
props Object Optional Parameters for the Object Store
index Object Object of the indexes to declare for the Object Store. Each index is a JSON object with the following properties:
index[...].name String Name for the Index
index[...].field String Field on the store to apply the index to
index[...].props Object Index properties object
