Class: SessionStorage

SessionStorage(configurationopt)

The Session Storage Adapter provides the Space Class the ability to interact with the sessionStorage api found in most modern browsers.

new SessionStorage(configurationopt)

Create a new SessionStorage. If no configuration is provided, the SessionStorage global object is used.
