Class: LocalStorage

LocalStorage(configurationopt)

The Local Storage Adapter provides the Space Class the ability to interact with the localStorage api found in most modern browsers.

Constructor

new LocalStorage(configurationopt)

Create a new LocalStorage. If no configuration is provided, the LocalStorage global object is used. The LocalStorage Adapter can provide independency by store name and space name.
Parameters:
Name Type Attributes Default Description
configuration Object <optional>
 {name = '', version = '', store = ''} Configuration Object for the Adapter
Properties
Name Type Description
name string Name of the Space
version string Version of the Space in Semantic versioning syntax
store string Name of the Object Store to use
